# MD Examples

This folder contains example markdown files demonstrating MD's rendering capabilities.

## Files

| File | Description |
|------|-------------|
| [syntax-highlighting.md](syntax-highlighting.md) | Code blocks with syntax highlighting for 16+ languages |
| [mermaid-diagrams.md](mermaid-diagrams.md) | Mermaid diagrams: flowcharts, sequence, ER, state, class |
| [ascii-diagrams.md](ascii-diagrams.md) | ASCII box-drawing diagrams (same syntax as Mermaid) |
| [tables.md](tables.md) | GitHub-flavored markdown tables |
| [formatting.md](formatting.md) | Text formatting, lists, blockquotes, links |
| [kitchen-sink.md](kitchen-sink.md) | Everything combined in one document |

## Testing

These examples can be used to test:

1. **MD Desktop App** - Open any file with `md <file>`
2. **MD Browser Extension** - View raw URLs on GitHub:
   - https://raw.githubusercontent.com/dnl-fm/md/main/docs/examples/mermaid-diagrams.md
   - https://raw.githubusercontent.com/dnl-fm/md/main/docs/examples/ascii-diagrams.md
