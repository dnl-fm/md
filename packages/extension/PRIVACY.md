# MD Extension Privacy Policy

**Last updated:** December 15, 2025

## Overview

MD is a browser extension that renders markdown files beautifully. We are committed to protecting your privacy.

## Data Collection

**We do not collect any personal data.**

The MD extension:
- Does NOT collect or transmit any user data
- Does NOT track your browsing activity
- Does NOT use analytics or telemetry
- Does NOT require user accounts

## Local Storage

The extension stores the following preferences locally in your browser using `chrome.storage.local`:
- Theme preference (dark/light)
- Font size setting
- Full width toggle state
- Last seen version (for update notifications)

This data never leaves your device.

## External Services

The extension loads syntax highlighting (Shiki) and diagram rendering (Mermaid) libraries from esm.sh CDN on-demand. No user data is sent to these services.

## Permissions

The extension requires these permissions:
- **Host permissions** (`*://*/*.md`): To detect and render markdown files on any website
- **Storage**: To save your preferences locally

## Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected in the "Last updated" date.
