Documentation

A fast, lightweight Markdown viewer for Linux, macOS, and Windows

Quick Start

# Download from GitHub Releases
# or build from source:

git clone https://github.com/dnl-fm/md.git
cd md
bun install
make build

Opening Files

  • Drag and drop a .md file onto the window
  • Use Ctrl+O to open a file dialog
  • Pass a file path as command line argument: md /path/to/file.md

Features

Live Preview

Markdown is rendered in real-time with GitHub-flavored markdown support. Tables, task lists, strikethrough, and autolinks are all supported.

Edit Mode

Toggle edit mode with Ctrl+Space. The WASM-powered editor includes:

  • Syntax highlighting for markdown (headings, code, bold, links, lists)
  • Line numbers (toggle with Ctrl+L)
  • Undo/redo with Ctrl+Z / Ctrl+Y
  • Tab/Shift+Tab to indent/dedent lines
  • Wrap selection with quotes/brackets
  • Virtual scrolling for large files

Mermaid Diagrams

Create diagrams using Mermaid syntax in fenced code blocks:

```mermaid
flowchart LR
    A[Start] --> B{Decision}
    B -->|Yes| C[Action]
    B -->|No| D[End]
```

Diagrams automatically match your current theme colors.

Syntax Highlighting

Code blocks are highlighted with Shiki for 16+ languages:

  • SQL, JavaScript/TypeScript, PHP, Python, Rust, Go, Bash, JSON
  • HTML/XML/SVG, CSS/SCSS/SASS, YAML, TOML, C/C++, Java/Kotlin, Ruby, Docker

Table of Contents

Press Ctrl+G to open the table of contents panel. Click any heading to navigate to it. The current section is highlighted as you scroll.

In-Document Search

Press Ctrl+F to search within the document:

  • Matches highlighted in yellow, current match in orange
  • Navigate with Enter/Shift+Enter or arrow buttons
  • Minimap on right edge shows all match positions

Print & PDF Export

Press Ctrl+P to print or export as PDF. The output is optimized with:

  • Proper page breaks
  • Syntax highlighting preserved
  • Relative images embedded as base64

File Watching

MD automatically reloads when the file changes on disk. Perfect for previewing while editing in your favorite editor (VS Code, Vim, etc.).

Drafts

Press Ctrl+N to create a new untitled draft. Drafts are auto-numbered (Untitled-1, Untitled-2, etc.) and can be saved to disk with Ctrl+S.

Configuration

Configuration is stored in:

  • Linux: ~/.config/com.fightbulc.md-preview/config.json
  • macOS: ~/Library/Application Support/com.fightbulc.md-preview/config.json
  • Windows: %APPDATA%\com.fightbulc.md-preview\config.json

Settings

Open settings with Ctrl+, to customize:

  • Font family and size (UI and markdown)
  • Theme colors (background, text, accent, etc.)
  • Mermaid diagram colors

Browser Extension

MD also includes a browser extension for Chrome and Firefox that renders raw .md URLs.

Supported URLs

  • raw.githubusercontent.com/*
  • gist.githubusercontent.com/*
  • GitLab/Gitea raw file URLs
  • Any URL ending in .md or .markdown served as text/plain

Tech Stack

  • Tauri 2 — Native app framework
  • SolidJS — Reactive UI framework
  • Rust/WASM — Editor backend
  • Shiki — Syntax highlighting
  • markdown-it — Markdown parser
  • Mermaid — Diagram rendering
  • Vite + Bun — Build tooling
