Changelog

All updates and improvements to MD

v0.9.0

Dec 14, 2025

  • Mermaid diagrams: Full support for flowcharts, sequence diagrams, class diagrams, and more
  • Theme-aware diagrams: Diagrams automatically match your dark/light theme colors
  • Table of Contents: Quick navigation panel (Ctrl+G) with current section highlighting
  • Live theme preview: See color changes instantly in the settings modal
  • New shortcuts: Ctrl+G (TOC), Ctrl+P (Print), Ctrl+H (Help), Ctrl+[/] (prev/next file)
v0.8.0

Dec 13, 2025

  • Language-aware syntax highlighting: Code blocks for 16+ languages (SQL, JS/TS, PHP, Python, Rust, Go, and more)
  • Editor ↔ Preview scroll sync: Position preserved when switching modes
  • MarkdownIt integration: Replaced marked with markdown-it for better extensibility
v0.7.0

Dec 13, 2025

  • WASM-based editor: Rust/WebAssembly editor replacing textarea
  • Ropey-backed text buffer for O(log n) edits on large documents
  • Built-in markdown syntax highlighting (headings, code, bold, links, lists)
  • Virtual scrolling for performance with large files
  • Undo/redo stacks managed in Rust
  • History validation: non-existent files removed from sidebar on startup
v0.6.0

Dec 2, 2025

  • Print / PDF export: via Ctrl+P or "Print" button
  • Print-optimized stylesheet with proper page breaks and sizing
  • Syntax highlighting preserved in PDF output
  • Relative images automatically embedded as base64
v0.5.0

Dec 2, 2025

  • JSDoc documentation across entire codebase
  • Per-file undo/redo stacks (history preserved when switching files)
  • Keyboard navigation in confirm dialog (Tab, Enter)
  • Confirm dialog with custom labels ("Discard"/"Stay")
v0.4.0

Dec 1, 2025

  • Draft tabs: Create untitled drafts with Ctrl+N
  • Line numbers: Toggleable with Ctrl+L in edit mode
  • Tab/Shift+Tab to indent/dedent lines
  • Wrap selection with quotes/brackets
  • Undo/Redo support (Ctrl+Z / Ctrl+Y)
  • Quick access via Ctrl+1-9, Ctrl+[/]
  • Help modal: Ctrl+H with shortcuts and version info
v0.3.0

Nov 28, 2025

  • In-document search: Ctrl+F with highlighting
  • Navigate matches with Enter/Shift+Enter
  • Minimap on right edge shows match positions
v0.2.1

Nov 28, 2025

  • Single-instance support: Opening a file reuses existing window
v0.2.0

Nov 27, 2025

  • Welcome modal for first-time users
  • Demo markdown file for showcasing features
  • Logging to ~/.md/md.log
  • Ctrl+W to close window
  • Tabbed settings modal with color customization
v0.1.0

Nov 27, 2025

  • Initial release
  • Markdown rendering with syntax highlighting
  • Light and dark themes
  • Edit mode with live preview
  • File open/save functionality
