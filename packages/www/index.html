Frequently asked questions

What platforms does MD run on?

MD runs on Linux, macOS, and Windows. It's built with Tauri, using native webviews for minimal footprint.

Is MD free?

Yes, MD is free and open source under the MIT license.

How do I open files?

Drag and drop files onto the window, use Ctrl+O to open a file dialog, or pass a file path as a command line argument.

Does MD watch files for changes?

Yes, MD automatically reloads when the file changes on disk, making it perfect for previewing while editing in your favorite editor.
